# Insurance Charges Prediction — Linear Regression Model

This project explores the classic *insurance.csv* dataset and builds a clean, interpretable Linear Regression model to predict medical insurance charges based on age, BMI, smoking status, region, and other personal attributes.

The aim is simple: to understand the factors that influence medical costs and to create a transparent model that explains these relationships clearly.

---

## 📊 Project Overview

The project includes:

- A full exploratory analysis of the dataset  
- Preprocessing using one‑hot encoding for categorical features  
- A Linear Regression model built through a clean Scikit‑Learn pipeline  
- Performance evaluation using MAE, RMSE and R²  
- A clear and readable code structure suitable for learning and portfolio use  

---

## 🧠 Key Insights

- Smoking status has the strongest influence on insurance charges  
- Age and BMI show a steady positive relationship with cost  
- Linear Regression provides a transparent baseline model  
- The dataset is well‑structured and requires minimal cleaning
